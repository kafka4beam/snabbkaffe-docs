    View Source snabbkaffe_nemesis (snabbkaffe v1.0.7)

    This module implements "nemesis" process that injects faults into system under test in order to test its fault-tolerance.

    usage

     Usage

    Somewhere in the tested code

      ?maybe_crash(kind1, #{data1 => Foo, field2 => Bar})

    Somewhere in the run stage

      ?inject_crash( #{?snk_kind := kind1, data1 := 42}
               , snabbkaffe_nemesis:always_crash()
               )
